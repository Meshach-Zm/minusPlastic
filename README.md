## Directory Structure

-   `/frontend`: The Expo mobile application.

### How to Run the Backend ‚öôÔ∏è

1.  Navigate to the backend directory:

    ```bash
    ./mvnw spring-boot:run
    ```
3.  The API will be running on `http://localhost:8080`.


    ```bash
    npm install
    ```

    ```bash
    npx expo start
    ```
